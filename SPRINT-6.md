# Sprint 6 — Personalización y retención

## Incluye

- Favoritos persistentes mediante localStorage.
- Historial local de los últimos 30 cálculos.
- Página `/mis-herramientas` para consultar favoritos e historial.
- Botón de compartir con Web Share API y copia de enlace como alternativa.
- Ayudas contextuales debajo de los campos.
- Acceso a los últimos resultados desde cada calculadora.
- Los datos permanecen exclusivamente en el navegador del usuario.

## Comprobación

```powershell
npm.cmd install
npm.cmd run build
npm.cmd run dev
```
