# Sprint 3A — Arquitectura y base visual

Cambios reales incluidos:
- Sistema visual global mediante variables y componentes reutilizables.
- Cabecera fija y responsive con menú móvil.
- Footer profesional y reutilizable.
- Componentes `Container`, `SectionHeading`, `Breadcrumbs`, `CategoryCard` y `ToolCard`.
- Migas de pan visibles con `BreadcrumbList` Schema.org.
- Mejoras de accesibilidad: enlace para saltar al contenido, navegación semántica y reducción de movimiento.
- Portada reconstruida usando los componentes centrales.
- Páginas de categorías y herramientas unificadas con la misma arquitectura.
- Corrección del enlace de tarjetas de categorías.
- Mejoras de metadatos sociales y `SearchAction` en el esquema del sitio.

Este sprint no modifica todavía la lógica interna de las 50 calculadoras. Esa parte corresponde al Sprint 3C.
