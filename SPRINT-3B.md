# Calculaya — Sprint 3B

## Cambios reales incluidos

- Rediseño visual completo de la portada.
- Hero profesional con buscador central, accesos rápidos y estadísticas.
- Buscador inteligente con sugerencias instantáneas mientras se escribe.
- Atajo Ctrl/Cmd + K para enfocar el buscador.
- Nuevas tarjetas de categorías y herramientas con mejor jerarquía visual.
- Página de populares unificada con el nuevo diseño.
- Página de búsqueda reconstruida, incluidos estados sin resultados.
- Cabecera refinada y navegación responsive.
- Nueva sección de beneficios para reforzar confianza y claridad.
- Mejoras específicas para móvil.

Este sprint se concentra en diseño y experiencia. La lógica completa de calculadoras se aborda en Sprint 3C.
