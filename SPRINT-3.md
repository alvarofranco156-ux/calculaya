# Calculaya — Gran Sprint 3

Incluye:
- Rediseño profesional de las páginas de herramientas.
- Las 50 herramientas conectadas a cálculos reales, con validaciones.
- Botones para calcular, reiniciar y copiar resultados.
- Migas de pan visibles y `BreadcrumbList` Schema.org.
- `WebApplication` y `FAQPage` Schema.org en cada herramienta.
- Preguntas frecuentes, explicación y herramientas relacionadas.
- Buscador instantáneo con sugerencias por nombre, categoría y descripción.
- Página 404 propia.
- Mejoras responsive, accesibilidad y estilos.

## Instalación
Copia el contenido sobre tu proyecto actual, sustituyendo archivos. No borres tu carpeta `.git`.

```powershell
npm.cmd install
npm.cmd run dev
```

Para comprobar producción:

```powershell
npm.cmd run build
git add .
git commit -m "Gran Sprint 3: diseño, calculadoras y SEO"
git push
```
